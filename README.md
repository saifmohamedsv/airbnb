## Features

## Tech Stack

**Client:** Next.js, Tailwindcss

## Run Locally

Clone the project

```bash
  git clone https://github.com/saifmohamedsv/airbnb
```

Go to the project directory

```bash
  cd airbnb
```

Install dependencies

```bash
  npm Install --legacy-peer-deps
```

Start the server

```bash
  npm run start
```
